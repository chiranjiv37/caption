# Implement Feature

You are implementing a new feature in Captions Studio.

Before writing code:

1. Search the repository for existing implementations.
2. Follow the existing architecture.
3. Reuse components, hooks, services, and utilities whenever possible.
4. Keep changes focused.

Implementation order:

1. Database (if required)
2. Backend models
3. Schemas
4. Services
5. Routers
6. Frontend API
7. Hooks
8. Components
9. UI

Never duplicate business logic.

Keep API responses consistent.

Explain any architectural decisions before introducing a new pattern.
