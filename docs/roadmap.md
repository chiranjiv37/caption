* caption editing
* transcription
* translation
* AI dubbing
* media management

# Guiding Principles

Every feature should improve one or more of

* speed
* usability
* reliability
* scalability
* maintainability

Avoid adding features that increase complexity without meaningful value.

---

# Success Metrics

Stable APIs
